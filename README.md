📖 About <a name="about"></a>

Neark is a location-based social planning application designed to streamline the process of meeting up with friends.

Unlike traditional "Find My Friends" style apps, Neark focuses on active, spontaneous plan creation. It bridges the gap between knowing where your friends are and actually making plans to see them. Whether you are looking for a quick coffee, a study session, or after-work drinks, Neark facilitates hyper-local coordination with granular control over who joins and how the bill is split.

✨ Key Features <a name="features"></a>

🎯 "Pick the Vibe" Creation: Quickly categorize plans (e.g., Cafe Time, Drinks, Beach, Ice Skate) or create custom activities.

💸 Split & Filter Logic: Set clear expectations upfront—define payment splits (50/50, I'll pay), age ranges, and group sizes before the plan even starts.

📍 Interactive Map Discovery: Toggle between card views and zoomed map views to see friends and activities nearby.

👻 Privacy Control: Includes "Ghost Mode," variable discovery radius, and specific profile visibility settings.

💬 Context-Aware Chat: Integrated messaging that embeds plan details directly into the conversation flow.

🛣️ Roadmap <a name="roadmap"></a>

Based on our current design specs and UI flow:

Phase 1: The "Plan" Engine 🏗️

[ ] Vibe Selector UI: Implementation of the grid selection (Cafe, Drinks, Beach, etc.).

[ ] Smart Filters: Logic for Age Range sliders, Gender preference, and Relationship status toggles.

[ ] Logistics Module: Time picker, Duration selector, and "Payment Split" logic (50/50 vs. Treat).

Phase 2: Map & Discovery 🗺️

[ ] Dual View: Switching between "Cards View" (swipeable list) and "Map View" (pins).

[ ] Nearby Suggestions: Integration with places API to suggest venues (e.g., "Zing Robot", "Panda Wok") based on location.

[ ] Privacy Core: Implementation of "Ghost Mode" and customizable Discovery Radius sliders.

Phase 3: Social Coordination 🤝

[ ] Request Flow: Sending "Join Requests" to specific plans.

[ ] Counter-Proposals: The "Something in mind?" screen allowing users to suggest alternative venues/times before accepting.

[ ] Confirmation Logic: "You're all set!" success states and Calendar API integration ("Add to Calendar").

Phase 4: Communication & Profile 💬

[ ] Embedded Chat: Real-time messaging with the "Plan Card" pinned at the top of the chat.

[ ] User Profiling: Interests tagging (Art, Sports, Pop Culture) and Smoking/Drinking preference toggles.
